Year 7 · Analysing & displaying data

Practice: Averages & Range

15 questions on the mode, median, mean and range, based on Unit 1.1. These are fresh questions written for self-practice — not copied from the textbook — covering finding each average, comparing data sets, and reasoning problems. Work each one out on paper, then reveal the worked answer to check.

15 questions · with worked answers
1
Mode

Find the mode of this set of numbers: 6, 3, 7, 3, 9, 3, 6, 1

Answer

Count how often each value appears. The number 3 appears three times, more than any other, so the mode is 3.

2
Mode · non-numerical data

A class chooses their favourite colour. Find the modal colour: red, blue, red, green, blue, red, blue, blue, green

Answer

Blue appears four times, red three times, green twice. The most common — the modal value — is blue.

3
Median · odd number of values

Work out the median of: 15, 4, 9, 2, 11, 6, 8

Answer

Put the values in order: 2, 4, 6, 8, 9, 11, 15. There are 7 values, so the middle (4th) value is the median: 8.

4
Median · even number of values

Work out the median of: 20, 7, 13, 5, 9, 16

Answer

In order: 5, 7, 9, 13, 16, 20. There are 6 values, so the median is halfway between the two middle ones: (9 + 13) ÷ 2 = 11.

5
Range

Work out the range of: 34, 12, 27, 5, 19

Answer

Range = largest − smallest = 34 − 5 = 29.

6
Mean

Work out the mean of: 7, 2, 9, 4, 8

Answer

Add the values: 7 + 2 + 9 + 4 + 8 = 30. Divide by how many there are: 30 ÷ 5 = 6.

7
Mean · decimals

Work out the mean of: 1.2, 0.8, 1.5, 0.5

Answer

Add the values: 1.2 + 0.8 + 1.5 + 0.5 = 4.0. Divide by 4: 4.0 ÷ 4 = 1.0.

8
Mean · real context

A shop records the number of customers each day for a week:
12, 9, 15, 11, 8, 14, 15. Work out the mean number of customers per day.

Answer

Total = 12 + 9 + 15 + 11 + 8 + 14 + 15 = 84. There are 7 days, so the mean = 84 ÷ 7 = 12 customers per day.

9
All four measures

For this set of marks, find the mode, median, range and mean: 8, 5, 5, 2, 10

Answer

In order: 2, 5, 5, 8, 10.
Mode = 5 (appears twice). Median = middle value = 5. Range = 10 − 2 = 8. Mean = (2 + 5 + 5 + 8 + 10) ÷ 5 = 30 ÷ 5 = 6.

10
Comparing data sets

Two netball players record goals scored over 5 matches.
Aisha: 3, 4, 2, 4, 2
Bea: 0, 7, 1, 0, 7
Find the mean and range for each, then say who is the more consistent scorer.

Answer

Aisha: mean = 15 ÷ 5 = 3, range = 4 − 2 = 2.
Bea: mean = 15 ÷ 5 = 3, range = 7 − 0 = 7.
Both have the same mean (3), but Aisha’s range is much smaller, so her scores vary less. Aisha is the more consistent scorer.

11
Reasoning

Write down a set of five numbers that has two modes.

Answer

Any set where two different values each appear most often works. For example 2, 2, 5, 7, 7 — here both 2 and 7 appear twice, so the set has two modes (it is ‘bimodal’).

12
Reasoning

Write down a set of numbers that has no mode, and explain why it has none.

Answer

For example 1, 3, 5, 8, 9. Every value appears exactly once, so no value is more common than the others — there is no mode.

13
Problem-solving

The mean of 5, 8, 6 and n is 7. Work out the value of n.

Answer

There are 4 values with a mean of 7, so their total must be 4 × 7 = 28. The three known values add to 5 + 8 + 6 = 19. So n = 28 − 19 = 9.

14
Problem-solving

Find a set of three numbers with a range = 0 and a mean = 5.

Answer

A range of 0 means the largest and smallest values are equal, so all three numbers must be the same. For the mean to be 5, each one must be 5: 5, 5, 5.

15
Reasoning · outliers

Seven friends compare how much pocket money they get each week:
£5, £5, £6, £5, £6, £5, £40.
One value is much larger than the rest. Is the mean or the median a better way to describe a ‘typical’ amount? Explain.

Answer

Mean = (5 + 5 + 6 + 5 + 6 + 5 + 40) ÷ 7 = 72 ÷ 7 ≈ £10.30. In order the values are 5, 5, 5, 5, 6, 6, 40, so the median = £5. The single large value (£40) is an outlier that drags the mean up well above what most friends actually get. The median (£5) is the better description of a typical amount.
